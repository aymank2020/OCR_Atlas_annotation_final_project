# Domain Rules: Kitchen / Cooking / Food Preparation
# Source: Atlas Training Hub + Discord + Culinary Conventions
# Priority: 3 (below Discord Golden Rules and Technical Glossary)

## Verb Precision
- slice: cutting food into pieces with a blade (knife through vegetable)
- cut: general severing. Use "slice" for food, "cut" for non-food (tape, packaging)
- chop: rougher cutting, larger pieces. Distinct from "slice" (finer pieces)
- dice: cutting into small cubes. Only if clearly dicing pattern visible
- pour: liquid from container to container
- stir: circular motion in liquid/mixture with utensil
- mix: combining ingredients. Coarse for "stir ingredients" if multiple steps
- peel: removing skin from fruit/vegetable (potato, banana)
- wash: running water over food/utensil
- drain: removing liquid from container (colander, sink)
- grease: applying oil/butter to surface (pan, baking sheet)

## Object Naming (Kitchen)
- "knife" is sufficient (not "chef's knife" or "paring knife" unless disambiguating)
- "pan" for any flat cooking vessel; "pot" for any deep cooking vessel
- "bowl" for any mixing/serving container
- "spoon" for any spoon (not "tablespoon" unless measuring context)
- "spatula" for any flat turning/scraping tool
- "cutting board" not "chopping board"
- Use food names when visible: "tomato", "onion", "chicken". If unclear: "food", "ingredient"

## Common Edge Cases (Kitchen)
- Tasting: do NOT label (not a task-relevant hand-object interaction)
- Reading recipe: do NOT label (no hand-object interaction)
- Turning stove on/off: "turn on stove" or "adjust stove" (not "inspect stove")
- Timer: "set timer" only if hands touch device
