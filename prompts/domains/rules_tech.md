# Domain Rules: Technology / Electronics / Repair
# Source: Atlas Training Hub + Discord + Technical Manual Conventions
# Priority: 3 (below Discord Golden Rules and Technical Glossary)

## Verb Precision
- loosen: turning/unfastening with a tool (screws, bolts, connectors). Use INSTEAD of "remove" when unscrewing.
- remove: physically detaching and taking away (battery from bay, cable from port). Post-loosen action.
- disconnect: separating cable or connector from port. Do NOT use "remove" for cables.
- insert: placing object INTO another (cable into port, screw into hole).
- press: applying force to button or contact point.
- peel: separating adhesive-backed components.

## Object Naming (Tech)
- Use manufacturer-neutral terms unless clearly identifiable:
  - "circuit board" not "PCB" or "motherboard" (unless clearly a motherboard)
  - "connector" not "FPC connector" or "ribbon cable connector" (unless visually certain)
  - "screw" is acceptable; "Phillips screw" or "Torx screw" ONLY if screw head is clearly visible
  - "battery" for any rechargeable cell
  - "screen" or "display" for any screen assembly
  - "cable" for any wire/ribbon connection

## Common Edge Cases (Tech)
- ESD strap: label "put on strap" or "wear strap" (not "ground yourself")
- Static mat: only label if hands interact with it
- Magnifying lamp: do NOT label looking through it (not a hand action)
- Heat gun: "apply heat to [object]" or "heat [object]"
- Tweezers: "pick up [object] with tweezers" (dense) or "remove [object]" (coarse)
