# CORE RULES (from Official Atlas Learning Hub — ALL 9 pages):

## 1. Core Mental Model
- Episode = full video task. Segment = continuous time span + one label.
- A segment = one continuous interaction with a primary object toward a single goal.
- Segment starts when hands engage, ends when hands disengage or goal changes.
- No hand contact → "No Action" (default).
- Do NOT label: walking, navigation, idle gestures, looking/examining, unrelated actions, "reach".
- Do NOT use numerical characters (1, 2, 5th). Use words ("three") or omit.

## 2. Label Format
- Imperative voice: "pick up spoon", "place box on table".
- Consistency: same verbs/nouns throughout episode. Don't alternate "wash"/"clean" without reason.
- Separators: multiple actions → comma or "and". NEVER omit separators.
  - ✅ "pick up cup, place cup on table" ✅ "pick up cup and place cup on table"
  - ❌ "pick up cup place cup on table"
- No numerals: ❌ "pick up 3 knives" → ✅ "pick up knives" or "pick up three knives".
- No intent language: label the physical action, NOT mental state.
  - ❌ "prepare to cut tape" → ✅ "pick up scissors", "cut tape".

## 3. Dense vs. Coarse
- A segment is either Dense OR Coarse — NEVER mix within a single segment (AUDIT FAIL).
- A video may contain both dense and coarse segments.
- Use coarse when: clear goal exists, atomic steps risk hallucination, or too many to list safely.
- Use dense when: multiple distinct hand actions needed, no single goal verb fits.
- Dense is NOT "better" — coarse is often preferred for accuracy.
- Soft limit: ≤20 words or ≤4 atomic actions per label.
- Accuracy and completeness always take priority over label length.

## 4. Verb Rules
- **FORBIDDEN**: "inspect", "check" — NEVER use.
- **FORBIDDEN**: "reach" — only if action is truncated at episode end AND no better verb exists.
  - If "reach" feels necessary, timestamps are probably wrong. Fix timestamps instead.
- Verb definitions:
  - pick up: object leaves surface/container (required in dense when pickup occurred)
  - place: object contacts surface and is released (required in dense when placement occurred)
  - move: coarse relocation = "pick up + place" as one goal. Allowed coarse substitute.
  - adjust: small corrective position change. Use instead of inspect/check.
  - hold: maintain grip without relocating. Only if task-relevant.
  - grab: grip itself is meaningful. Rare, use sparingly.
  - loosen: turning/unfastening with a tool. Use instead of "remove" for unscrewing.
  - remove: physically detaching and taking away an object.
- Every verb MUST have an object: ❌ "pick up, place on table" → ✅ "pick up cup, place cup on table".

## 5. HOLD RULE (Critical — Discord 2026-03-26)
- "hold" MUST be labeled in EVERY segment where ego holds an object.
- When ONLY holding (no other action): use "hold [object]" ONLY.
- NEVER add intent/purpose: ❌ "hold phone to check charging status" → ✅ "hold phone".
- If held object is already in another action in the same segment → do NOT add separate hold.
  - ✅ "hold cup, pick up cloth" (cup NOT in "pick up cloth")
  - ✅ "wipe cup with cloth" (cup IS in action → no "hold cup" needed)
  - ❌ "hold cup, wipe cup with cloth" (cup redundant)

## 5.5 Handling Edge Cases (Blurry, Hidden, Unclear Objects)
- NEVER invent or guess a specific object name if you cannot clearly see it.
- NEVER use placeholder tags like [UNCERTAIN], [UNKNOWN], or [OCCLUDED].
- MUST fallback to broad categorical nouns (Hypernyms):
  - Unknown tool -> "tool"
  - Unknown part -> "part" or "component"
  - Unknown cable -> "cable" or "wire"
  - Unknown container -> "container"
  - Completely unidentifiable -> "item" or "object"
- Examples:
  - Video: hand reaches for small metal object, unclear in frame.
    BAD: "pick up M4 screw" (hallucinated specificity)
    BAD: "pick up [UNCERTAIN] object" (placeholder tag)
    GOOD: "pick up part" or "pick up item"
  - Video: person grips something cylindrical, blurry motion.
    BAD: "pick up Phillips screwdriver" (guessing)
    GOOD: "pick up tool"

## 6. No Action Rules
- Use "No Action" ONLY when: hands touch nothing, OR ego is idle/irrelevant behavior.
- Do NOT split solely to isolate "No Action" pauses.
- Do NOT combine "No Action" with real actions in one label (AUDIT FAIL).
- Do NOT use "No Action" if ego is holding a task-relevant object → use "hold [object]".

## 7. Object Naming
- Identify only what you can defend. If unsure → "tool", "container", "cloth".
- Fix incorrect object naming (spoon vs fork).
- Stay consistent through the episode.
- Adjectives only to disambiguate: "blue cloth" vs "white cloth". If only one → "cloth".
- "place" requires a location: "place cup on table", "place cup in bin".
- Left/Right allowed from ego perspective but not required.
- Avoid body parts unless unavoidable: ✅ "wash spoon" not "wash spoon with hand".

## 8. Segment Editing
- Timestamps: Start = hands begin engaging. End = hands disengage.
- Minor idle time inside segment is OK if still one continuous interaction.
- Don't extend into a new action. Don't cut off completion.
- MERGE only if: same action/goal AND hands NEVER disengage AND combined ≤ 10s.
- Preferred segment duration: 2-5 seconds ("Sweet Spot"). Avoid segments > 10s.
- Do NOT merge: repeated pick up → place cycles with disengagement, or different objects/goals.
- SPLIT when: hands disengage + new interaction, or new goal begins.
- If continuous action > 10s: split and use "continue [verb] [object]" or repeat the same label. Aim for 2-5s density.

## 9. Repeated Actions
- Ego disengages and repeats → separate segments.
- Ego never disengages → one segment (often coarse).
- Simultaneous actions: use coarse goal label OR dense enumerated actions. Do NOT invent steps.

## 10. Audit Fail Conditions
A segment FAILS if ANY are true:
1. Missed major task-relevant hand action.
2. Hallucinated (non-occurring) action/object.
3. Timestamps cut off action or include different action.
4. Forbidden verbs used ("inspect/check", "reach" except truncated-end).
5. Dense/coarse mixed in one label.
6. "No Action" combined with action.

## 11. Ideal Segment Checklist
✅ One goal ✅ Full action coverage ✅ Accurate verbs ✅ No hallucinated steps ✅ Dense OR coarse (not mixed)
Quality over quantity. A well-labeled segment accurately captures the main hand-object interaction from start to finish, using clear and consistent language.

## 12. Simplified Technical English (STE) Compliance
- Label sentences MUST be ≤20 words.
- Use imperative voice: "pick up tool" NOT "the tool is being picked up".
- Replace vague verbs: "make sure" → "verify"; "do" → specific verb.
- One instruction per sentence. Never chain >2 actions without comma separator.
- Technical terms MUST stay consistent throughout entire episode (no synonyms).
- Forbidden ambiguous terms: "stuff", "things", "it", "that" → use actual object name.
